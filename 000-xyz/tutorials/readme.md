# Dart 内置库

包名 描述

dart:async 异步编程，提供Future,Stream类
dart:collection 集合
dart:convert 不同类型的字符编码解码
dart:core Dart语言核心功能，内置类型
dart:html 网页开发用到的库
dart:io 文件读写，IO相关
dart:math 数字常量及函数，随机算法等
dart:svg 事件和动画矢量图支持
